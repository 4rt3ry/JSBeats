# JSBeats

Test your ability to keep in sync with complex rhythms. 

## Code Conventions

### HTML
`ID`, `class`, `file names`

-  lowercase-dashes

### JavaScript
- All scripts should be in modular scope
- Use of `var` is prohibited
- All utility functions must be pure functions

`let`, `const` 

- lowerCamelCase

`class`

- UpperCamelCase
